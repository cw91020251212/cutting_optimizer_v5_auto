
鋁材切割排

使用說明
設置：選擇語言和顯示單位，輸入鋸片厚度
切割項目：輸入需要切割的長度、數量和用途，點擊"新增項目"添加到列表
編輯項目：點擊表格中的"編輯"按鈕可將項目回填到輸入欄進行修改，然後點擊"更新項目"保存
材料：輸入可用材料的長度和數量，點擊"新增材料"添加到列表
編輯材料：點擊表格中的"編輯"按鈕可將材料回填到輸入欄進行修改，然後點擊"更新材料"保存
計算：點擊計算按鈕獲取最佳切割方案
結果：查看切割方案詳情和圖形化展示
可以儲存和載入常用方案
點擊"列印報告"可列印當前方案
提示：

使用公制單位時輸入米/厘米/毫米
使用英制單位時輸入尺/寸/分
鋸片厚度會影響材料利用率
系統會自動計算餘料並優化方案
智能優化方案會將相同尺寸的切割件盡量排列在一起，提高開料效率
點擊"清除輸入"按鈕可以快速清空輸入欄
新增或更新項目後，輸入欄會自動歸零，方便輸入下一個項目
編輯項目時，輸入欄的單位會跟隨該項目原本的單位（公制/英制）
